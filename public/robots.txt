# =============================================================
# robots.txt：搜索引擎爬虫访问规则
# =============================================================
#
# 【泛化描述】robots.txt 是告诉搜索引擎爬虫哪些页面可以/不可以抓取的文件。
#
# 【字段含义】
#   User-agent: *      → 规则适用于所有爬虫（* 是通配符）
#   Disallow:          → 允许所有爬虫访问所有内容（留空表示无限制）
#
# 【典型场景】
#   网站上线后 → 搜索引擎爬虫（如 Googlebot）会先读取 robots.txt
#   → 根据规则决定抓取哪些页面

# https://www.robotstxt.org/robotstxt.html
User-agent: *
Disallow:
